# Student Club Website

## Description

This is a simple React application created as an assignment.

The website contains:
- Home Page
- Team Page
- Navigation Bar
- Active Navigation Highlight

## Technologies Used

- React
- Vite
- React Router DOM
- CSS

## Project Structure

```
student-club/
│
├── src/
│   ├── components/
│   │   └── Navbar.jsx
│   ├── pages/
│   │   ├── Home.jsx
│   │   └── Team.jsx
│   ├── App.jsx
│   ├── main.jsx
│   └── index.css
│
├── public/
├── package.json
├── vite.config.js
└── README.md
```

## Installation

Install all dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Open in your browser:

```
http://localhost:5173
```
